# DomnerTech.CATemplate (mnt-ca)
## To install this project template (path: base project template)

```sh
dotnet new install .
dotnet new install . --force
```

## To uninstall this project template (need this when have something update) `dotnet new uninstall [path to the project template]`
```sh
dotnet new uninstall {path}\DomnerTech.CATemplate
```

## How to use `dotnet new [template name] -n [project name]`

```sh
dotnet new mnt-ca -n ProjectName
dotnet new mnt-ca -o ProjectName
```
